API Documentation

OpenAI-compatible DeepSeek API Gateway. Drop-in replacement for OpenAI SDK.

Base URL

http://65.49.201.211/v1

All endpoints below are relative to this base URL.

Available Models

Model IDDescriptionPlans
deepseek-v4-flashFast, affordable chat modelAll plans
deepseek-v4-proReasoning-heavy tasks, higher qualityPro only

Chat Completions

# curl
curl http://65.49.201.211/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"deepseek-v4-flash","messages":[{"role":"user","content":"Hello!"}],"max_tokens":100}'

Python (OpenAI SDK)

from openai import OpenAI

client = OpenAI(
    base_url="http://65.49.201.211/v1",
    api_key="YOUR_KEY",
)

response = client.chat.completions.create(
    model="deepseek-v4-flash",
    messages=[{"role": "user", "content": "Hello!"}],
    max_tokens=100,
)
print(response.choices[0].message.content)

Node.js

import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "http://65.49.201.211/v1",
  apiKey: "YOUR_KEY",
});

const response = await client.chat.completions.create({
  model: "deepseek-v4-flash",
  messages: [{ role: "user", content: "Hello!" }],
  max_tokens: 100,
});
console.log(response.choices[0].message.content);

List Models

curl http://65.49.201.211/v1/models \
  -H "Authorization: Bearer YOUR_KEY"

Returns only the models your plan has access to.

Key Usage

curl http://65.49.201.211/v1/key/usage \
  -H "Authorization: Bearer YOUR_KEY"

Returns your current token usage, quota, rate limit, and plan details.

Error Codes

CodeMeaning
401Invalid or missing API key
402Token quota exceeded. Top up to continue.
403Key disabled, or your plan does not include this model
429Rate limit exceeded. Wait and retry.
502Upstream error. The service may be temporarily unavailable.

Streaming

Set "stream": true in your request body. Works with all OpenAI SDK streaming methods.

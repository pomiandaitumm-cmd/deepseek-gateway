What is this?

A managed API endpoint that forwards your requests to DeepSeek's models using the OpenAI request/response format. If your code already works with OpenAI's Python SDK, it works here 鈥?change the base URL and key.

Quick Start

from openai import OpenAI

client = OpenAI(
    base_url="http://65.49.201.211/v1",
    api_key="sk-gateway-YOUR-KEY",
)

response = client.chat.completions.create(
    model="deepseek-v4-flash",

FAQ

Can I use the standard OpenAI Python SDK?

Yes. Set base_url to our endpoint and pass your gateway key. No custom client needed.

How are tokens counted?

We use the usage.total_tokens value returned by DeepSeek for each request. You can check your usage anytime with a simple API call.

What happens when I run out of tokens?

You'll receive an HTTP 402 response. Contact us to add more tokens to your key.

Is there a rate limit?

Yes. Each key has a configurable requests-per-minute limit. The default is 30 RPM. You'll get an HTTP 429 if exceeded.

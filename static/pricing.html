ModelRelay is a small independent gateway focused on one thing: making DeepSeek API accessible through PayPal with fixed budgets. No account system. No complexity. Buy a key, use it until the budget runs out, buy another. Direct DeepSeek and OpenRouter are excellent services — we are simply a different option for those who prefer PayPal checkout and per-key budget isolation.

Budget-Based Billing — You purchase an API budget (USD). Each request deducts from your budget based on real DeepSeek token pricing: input cache-miss $0.14/M, input cache-hit $0.014/M, output $0.28/M. Pro model costs more. Cache hits are 10x cheaper — use system prompts to save.
